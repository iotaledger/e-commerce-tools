# Troubleshooting

The best source for sorting out problems is the IOTA [Discord](https://discord.gg/iota). Join the [#iota-cloud-tools](https://discord.com/channels/397872799483428865/910192737842790400) channel and ask your questions there. Additionally, the [FAQ](./faq) might have the answer to your problem.

Please make sure to have the latest version of the Integration Services installed and all other packages up to date.
